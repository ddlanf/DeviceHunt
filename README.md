## Core Features
This apps allows the users to find specs and watch Youtube reviews of the devices they have searched. 
Users can filter searches by device brand, name, and model. In addition, users can limit the number 
of results desired.

### Technologies used
* Html
* CSS
* Javascript
* jQuery
